置顶与高亮 bgm.tv 首页今日放送的在看条目与集。

**油猴脚本链接:**
[bgmAirToday.user.js](https://raw.githubusercontent.com/kjtsune/UserScripts/main/bgmAirToday/bgmAirToday.user.js)

效果：

1. 今日播放的条目会在最前面。
2. 今日播放的集会变成高亮绿色。
3. 仅调整追更条目，即前一集是已播放状态。
